빌드 에러 수정 완료:

문제: 테스트 파일(*.test.ts, *.test.tsx)이 TypeScript 빌드에 포함되어 'global' 타입을 찾을 수 없어 에러 발생

해결:
- tsconfig.json 수정하여 테스트 파일을 빌드에서 제외
- include/exclude 패턴을 더 구체적으로 지정
- 빌드 테스트 통과 확인

수정 내용:
1. tsconfig.json에서 *.test.ts, *.test.tsx 파일을 exclude에 추가
2. include 패턴을 더 구체적으로 지정하여 소스 파일만 포함
3. 로컬 빌드 테스트 성공 확인 (npm run build)
